# Neurohelmet Homebrew Tap

Homebrew formulae for [Neurohelmet](https://github.com/tympaniplayer/neurohelmet) —
a keyboard-driven terminal BattleTech record-sheet tracker.

## Install

```sh
brew install tympaniplayer/tap/neurohelmet
```

Upgrade with `brew upgrade neurohelmet`.

## Notes

`Formula/neurohelmet.rb` is generated automatically on each Neurohelmet release by its
[release workflow](https://github.com/tympaniplayer/neurohelmet/blob/main/.github/workflows/release.yml).
Don't edit it by hand — it's overwritten on the next release.
